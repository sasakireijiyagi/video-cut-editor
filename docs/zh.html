如果 Mac 提示"无法打开"

由于未通过（付费的）Apple Developer Program 签名，macOS 首次运行时会显示警告。这不是检测到恶意软件，只是表示开发者身份未经认证。源代码已在 GitHub 完全公开，且不进行任何网络通信。

  1. 在警告窗口点击"完成"（不要点击"移到废纸篓"）
  2. 打开系统设置 → 隐私与安全性
  3. 向下滚动并点击"仍要打开"

Features

Transcription

高精度转写

搭载 OpenAI Whisper，支持 100 种语言 — 讲座、会议、采访均可使用。

Editing

轻松编辑 & 剪切

选择 SRT 片段，用 ffmpeg 导出剪切视频。支持精细调整开始/结束时间。

Batch

批量转写

将多个文件加入队列。处理途中也可继续添加文件，实时显示剩余时间。

Filler Cut

去除填充词

自动检测并去除 8 种语言的填充词。词表可自由定制。

Subtitle

字幕合成

将粗体白字黑边字幕直接烧录进视频。自动识别竖屏视频。

Export

EAF 导出

可导出为语言分析工具 ELAN 专用的 EAF 格式，适合学术研究使用。
