개발 지원 기부

Features

Transcription

고정밀 받아쓰기

OpenAI Whisper 탑재. 100개 언어 지원 — 강의, 회의, 인터뷰에 최적.

Editing

간편 편집 & 컷

SRT 세그먼트를 선택해 ffmpeg으로 컷 출력. 시작·종료 시간의 세밀한 조정도 가능.

Batch

일괄 받아쓰기

여러 파일을 대기열에 추가. 처리 중에도 파일 추가 가능. 남은 시간 표시.

Filler Cut

필러 단어 제거

8개 언어의 필러 단어를 자동 감지·제거. 단어 목록을 자유롭게 커스터마이즈.

Subtitle

자막 합성

굵은 흰 글씨에 검은 외곽선을 동영상에 직접 합성. 세로 동영상 자동 감지.

Export

EAF 내보내기

언어 분석 도구 ELAN용 EAF 파일로 내보내기. 연구 용도에 최적.
